## Installation

Please `cd` into project directory before running all below commands.

Install dependencies:

```bash
$ npm install
```

Create an .env file in the root folder and save the enviroment variables:

```bash
// mongoDB settings
MONGO_URL = ''
```

Run the server using the following command :

```bash
nodemon app.js
```
